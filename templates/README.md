
# Templates

In this directory you find sample files for configuring Mallob's job streaming.

* **tc.json** : Sample file and documentation on -client-template
* **tj.json** : Sample file and documentation on -job-template
* **list.txt** : Example file for -job-desc-template (simply one whitespace-separated list of files (all for the same job) per line)

There's also some additional template files, from past experiments, to illustrate some further use cases.
